# Overview
This library containing 7 popular background subtraction algorithms: 
* adaptive median filtering
* eigenbackground
* single Gaussian
* Gaussian mixture models
* adaptive Gaussian mixture models
* running mean
*  mediod filtering

This library makes use of [OpenCV](http://opencv.org/).

# Building with CMake

	$ mkdir _build
	$ cd _build
	$ cmake ..
	$ make
